# 竞品研究 Agent

## 何时使用
仅在 Deep 模式下使用。Quick 模式跳过此文件。

## 执行方式
使用 WebSearch 工具执行搜索。共 3 组搜索，每组围绕一个角度。
搜索完成后，汇总成一个结构化的竞品概览。

## 搜索策略

### 地区适配

搜索前先判断点子的目标市场：
- **面向中国用户**：增加中文搜索 + 国内平台搜索
- **面向海外用户**：使用英文搜索策略
- **用户没说 / 不确定**：中英文都搜，覆盖更全面

判断依据：用户的描述语言、点子名称、目标用户特征。

### 第一组：直接竞品
搜索该领域已有的直接竞品产品，了解市场现状。

搜索 query 示例（根据用户的点子替换关键词，按地区选择）：

**通用搜索：**
- `"{点子关键词} 产品 竞品 工具"`
- `"{idea keywords} alternatives competitors tools"`

**海外侧重：**
- `"site:producthunt.com {related keywords}"`
- `"{idea keywords} startups funding"`

**中国侧重：**
- `"site:36kr.com {点子关键词}"`
- `"site:sspai.com {点子关键词}"`
- `"{点子关键词} 竞品分析 国内"`

关注信息：
- 产品名称、官网
- 目标用户群体
- 核心功能
- 定价模式（免费/付费/订阅）
- 大致规模（用户量、融资、团队大小）

### 第二组：间接替代品
搜索用户当前解决同类问题的替代方案。

搜索 query 示例：
- `"{问题关键词} 解决方案 替代 {领域}"`
- `"how people solve {problem} without {your solution}"`
- `"{问题关键词} spreadsheet manual workflow"`

关注信息：
- 用户当前用什么方式解决这个问题
- 替代方案的优缺点
- 替代方案的价格（很多可能是免费的）

### 第三组：市场格局 + 死亡清单
搜索该领域的整体市场信息、趋势，**以及失败的产品**。

**市场格局搜索：**
- `"{领域} 市场规模 趋势 报告"`
- `"{domain} market size growth report"`
- `"{领域} 创业 投资 融资 {近2-3年}"`

**失败信号搜索（重要）：**
- `"{领域} 产品 关闭 停止运营 下架 倒闭"`
- `"{domain} app shutdown discontinued failed"`
- `"{领域} 创业失败 关门 原因 教训"`

关注信息：
- 市场规模和增长趋势
- 近期投融资动态
- 行业是否有垄断玩家
- 是否有明显的市场空白
- **死了多少产品？死因是什么？活下来的有什么共同特点？**

死亡清单特别重要——如果一个领域的死亡率异常高，
这本身就是最致命的市场信号，比市场规模数据更有价值。
