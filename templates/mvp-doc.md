# MVP 文档模板

Deep 模式的文件输出模板。保存为 MD 文件到工作目录。
分两层：核心层（始终输出）和扩展层（用户选择时附加）。
**输出语言跟随用户的输入语言。**

---

## 输出说明

Deep 模式 Phase 4 时，询问用户：
> "MVP 文档要**精简版**（核心评估 + 行动计划）还是**完整版**（含所有模型展开 + 竞品分析 + Agent 研究）？"

- **精简版**：只输出核心层
- **完整版**：输出核心层 + 扩展层

文件名格式：`idea-evaluation-{点子关键词}-{日期}.md`

## 要点

- 核心层控制在 100 行以内，用户能快速看完
- 扩展层按模型顺序展开，每个模型引用 Phase 3 的评估结果
- 竞品/案例部分直接引用 Phase 2 的 agent 输出
- 所有具体数字和分析结论必须来自搜索数据，不要编造
- {OPC: ... / 团队: ...} 的地方根据用户选择的视角填入对应内容
